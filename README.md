How to complie and run the program:
 -> Open a terminal/command prompt and navigate to the src directory.
 -> Use the following commands:
    -> To complie the program: javac FileName.java
    -> To run the program: java FileName
      Example: 
      -> javac Basics.java
      -> java Basics
